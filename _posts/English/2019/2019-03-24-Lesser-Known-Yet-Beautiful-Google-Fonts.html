Google Fonts is a library of open-source fonts, an interactive web directory for browsing the library, and APIs for conveniently using the fonts in websites and more. Google Fonts has now 915 fonts with more than 28 trillion views. Anyone can download any font and use it in his work.

On Google Fonts, there are some very popular fonts like Roboto, Open Sans, Lato, Montserrat, etc. For example, Open Sans was served by Google Fonts API around 30 billion times in the last week as it's stated on Google Fonts website today.

But I think that not all great fonts are popular. I built a web application (Pair & Compart) last year to make it easier to try and compare different Google fonts without downloading or installing them. I recently used it to choose a font for some parts of my resume. During the process, I discovered some beautiful fonts that I'd like to share them here.

For each font, I will list the font name, a picture showing the font in action with the following CSS properties:

font-size: 32px;
line-height: 2

, a link to the font page on Google Fonts where you can download the font or get a link for using it in your website, and some info or opinions about the font.
